== Description ==

This plugin prettifies/beautifies/highlights source code placed inside `<pre class="prettyprint">` tags. It is a light plugin with only 2 files.

New in 3.2.2 - use the hammer icon in the editor to paste your code and automatically have it converted to pretty code.

== Installation ==

1. Upload the plugin folder to your `/wp-content/plugins/` directory
2. Activate the plugin via the Plugins menu in WordPress
3. Add code between `<pre class="prettyprint">` tags
